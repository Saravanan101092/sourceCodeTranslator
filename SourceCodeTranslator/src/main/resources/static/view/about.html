Source code translator

Why this tool?

Translating documents using Google Translate has a number of issues

  • Google translated does not let us translate whole files at a time unless it is paid version.
  • It does not detect source code syntax such as camel case, underscores etc

Hence this tool intends to overcome those drawbacks

Some useful features:

  • Download translated files instantly
  • Keep track of all translated files
  • OpenOffice: text(odt), spread sheet(ods) can be translated.
  • MS Office: word(doc,docx) and excel files(xls,xlsx). (Translation in word files affect the styling for now. Will work on it.)
  • Flat files: txt, csv, sql, cs, c, java, php etc.
  • Code will detect camel casing and split the string before translating it. 

 

          

 

Note: Translation quality is only as good as Google translate. Could not be as good as getting it translated by human translator.
